التطبيقات

ثلاث تطبيقات لكل فريقك

كل عضو في فريقك له تطبيقه الخاص بالصلاحيات المناسبة

👔

تطبيق المدير

تحكم كامل في كل تفاصيل مشروعك من مكان واحد

  • لوحة تحكم شاملة بإحصائيات لحظية
  • إدارة القائمة والأصناف والفئات
  • متابعة الطلبات وحالاتها
  • إدارة الموظفين والحضور والرواتب
  • إدارة المخزون والمكونات
  • إدارة العملاء وسجل الطلبات
  • إدارة الطاولات
  • تحليلات وتقارير متقدمة
  • تصدير التقارير (PDF / Excel)
💳

تطبيق الكاشير

نقطة بيع سريعة وسهلة الاستخدام لتسجيل الطلبات

  • نقطة بيع (POS) سريعة وذكية
  • إنشاء طلبات (محلي - تيك أواي - توصيل)
  • إدارة الطاولات وحالاتها
  • متابعة الطلبات الحالية
  • طباعة الإيصالات ومشاركتها
  • تسجيل الحضور (PIN / QR)
  • تسجيل الأوفرتايم
  • ملخص الشيفت اليومي
  • دعم العمل أوفلاين
  • لوحة التوصيل
🛵

تطبيق التوصيل

تطبيق خاص بموظفي التوصيل لإدارة طلباتهم

  • استقبال طلبات التوصيل
  • تتبع حالة الطلبات
  • عرض تفاصيل العميل والعنوان
  • تحديث حالة التوصيل
  • سجل الطلبات السابقة
  • إشعارات فورية
المميزات

كل اللي محتاجه في مكان واحد

نظام متكامل يغطي كل جوانب إدارة مقهاك أو مطعمك

🏪

نقطة البيع (POS)

نظام بيع سريع وذكي مع دعم الأوفلاين

  • إنشاء طلبات بضغطة زر
  • دعم الفاريانت والإضافات
  • طرق دفع متعددة (كاش - كارت - محفظة)
  • خصومات يدوية وعروض ونقاط ولاء
  • دعم الضرائب
  • العمل بدون إنترنت
📋

إدارة الطلبات

تتبع كل طلب من الإنشاء حتى التسليم

  • حالات متعددة (جديد - قيد التحضير - جاهز - مكتمل)
  • قنوات متعددة (محلي - تيك أواي - توصيل)
  • استرجاع وإلغاء الطلبات
  • ربط الطلبات بالطاولات والعملاء
🍽️

إدارة القائمة

تحكم كامل في الأصناف والفئات

  • تصنيف الأصناف في فئات
  • نظام الفاريانت (أحجام - إضافات)
  • صور الأصناف
  • التحكم في التوفر
  • القائمة الرقمية (Digital Menu)
📦

إدارة المخزون

تتبع المكونات والمواد الخام بدقة

  • إضافة وخصم الكميات
  • تنبيهات المخزون المنخفض
  • ربط المكونات بالوصفات
  • خصم تلقائي عند البيع
  • سجل حركات المخزون
🧾

الوصفات (BOM)

ربط الأصناف بمكوناتها من المخزون

  • إنشاء وصفات لكل صنف
  • حساب التكلفة التلقائي
  • التحقق من توفر المكونات
  • خصم المكونات عند البيع
👷

إدارة الموظفين

إدارة شاملة لكل موظفينك

  • بيانات الموظفين والأدوار
  • تفعيل/تعطيل الموظفين
  • كروت الموظفين (QR Code)
  • رقم PIN لكل موظف
  • ربط الموظف بحساب مستخدم
⏰

نظام الحضور والانصراف

تتبع حضور موظفينك بدقة

  • تسجيل بـ PIN أو QR Code
  • تسجيل يدوي
  • حساب ساعات العمل
  • ملخص الحضور اليومي
  • دعم العمل أوفلاين
💰

الرواتب والأوفرتايم

نظام رواتب متكامل مع حساب الأوفرتايم

  • رواتب يومية أو شهرية
  • تسجيل ساعات الأوفرتايم
  • صرف الرواتب فردي أو جماعي
  • سجل المدفوعات
🪑

إدارة الطاولات

تنظيم وتتبع حالة الطاولات

  • إضافة طاولات مع السعة
  • حالات (متاحة - مشغولة - محجوزة)
  • ربط الطلبات بالطاولات
  • عرض شبكي سهل
👥

إدارة العملاء

بناء قاعدة عملاء قوية

  • بيانات العملاء وأرقام الهاتف
  • عناوين متعددة لكل عميل
  • سجل طلبات كل عميل
  • إحصائيات (إجمالي الطلبات والإنفاق)
  • سجل نقاط الولاء لكل عميل
  • سجل الخصومات لكل عميل
  • فلترة حسب حالة النقاط
⭐

نقاط الولاء

كافئ عملاءك وزوّد ولاءهم

  • كسب نقاط تلقائي مع كل طلب
  • استبدال النقاط كخصم
  • حد أدنى للاستبدال
  • سجل كامل لحركات النقاط
  • تحليلات أداء برنامج الولاء
🏷️

الخصومات اليدوية

مرونة كاملة في تطبيق الخصومات

  • خصم ثابت أو نسبة مئوية
  • سبب الخصم (اختياري)
  • حد أقصى للخصم بدون موافقة
  • موافقة المدير بـ PIN للخصومات الكبيرة
  • تحليلات استخدام الخصومات
🎁

إدارة العروض

أنشئ عروض جذابة لزيادة المبيعات

  • عروض خصم ثابت أو نسبة
  • تحديد تاريخ بداية ونهاية
  • تطبيق تلقائي أو بكود برومو
  • تفعيل وإيقاف العروض
  • تحليلات أداء كل عرض
🚚

نظام التوصيل

إدارة كاملة لعمليات التوصيل

  • لوحة تحكم التوصيل
  • تعيين موظفين للتوصيل
  • تتبع حالة التوصيل
  • رسوم التوصيل
  • تطبيق خاص للسائقين
🧾

الإيصالات الرقمية

إيصالات احترافية قابلة للمشاركة

  • إيصالات رقمية بتصميم احترافي
  • مشاركة عبر رابط
  • طباعة الإيصالات
  • QR Code لكل إيصال
🏛️

الضرائب

إدارة مرنة للضرائب

  • شرائح ضريبية متعددة
  • حساب الضريبة التلقائي
  • ضريبة افتراضية
📢

الإعلانات والتنبيهات

تواصل مع فريقك بسهولة

  • إنشاء إعلانات للموظفين
  • تتبع القراءة
  • عدد الإعلانات غير المقروءة
📄

التصدير والتقارير

صدّر بياناتك بالصيغة المناسبة

  • تصدير PDF احترافي
  • تصدير Excel للتحليل
  • تقارير بفترات زمنية
  • معاينة قبل التصدير
📱

متعدد المنصات

يعمل على كل الأجهزة

  • Android (موبايل وتابلت)
  • iOS (آيفون وآيباد)
  • Desktop (ويندوز وماك ولينكس)
  • بصمة وFace ID
